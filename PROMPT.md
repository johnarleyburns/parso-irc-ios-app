# Initial Propmt

create a plan for an award-winning ios iphone IRC chat client written in swift.  constraints are i don't have an apple developer account or a macbook, so you will need to use the 7-day self-signed model with a github mac runner for building, and testflight is preferred if possible.  I want at least libra.chat supported with the default channels of #libra and #dylt-dev (Daylight development), as well as any other very popular irc chats.  It is IMPORTANT that you spend a lot of time thinking about the UI/UX so it is compliant with all the latest apple style guidelines and has a very easy to use, seemless interface.  It should feel very natural to anyone who's used ios messaging apps.  For power users, manual irc commands such as /nick and /join should work, but for the most typical use cases they should not be required at all by typical users, who want a more intuitive graphical interface.  The design should be pixel-perfect and really an outstandard award-winning UI/UX.  Take your time and devise a perfect plan.
